Structural biology tools for analyzing three-dimensional structures of proteins, nucleic acids, and molecular complexes. Process cryo-EM data, predict structures, and refine atomic models.

::callout{color="green"}
**High-Performance Structure Analysis**: GPU-accelerated workflows for efficient cryo-EM processing and structure prediction.
::

## Overview

This collection includes workflows for:

- **Cryo-EM Processing** - Single particle analysis and reconstruction
- **Structure Prediction** - AI-powered protein structure modeling
- **Structure Refinement** - Model building and validation
- **Visualization** - Interactive molecular graphics
- **Data Analysis** - Resolution estimation and quality assessment

## Available Workflows

Browse the workflows in this category to find structural biology tools. Each workflow includes processing pipelines, parameter guides, and validation methods.

## Deploy

Deploy structural biology workflows:

```bash
# Deploy a cryo-EM or structure workflow
dxflow workflow create --identity <workflow-name> <compose-file>
dxflow workflow start <workflow-name>

# Follow the logs live
dxflow workflow logs --live <workflow-name>
```

Full desktop web browsers running on remote compute and streamed to your browser — useful for isolated browsing, automation, and testing. Each builds on the [Void Desktop](/hub/desktop/void) image and opens maximized.

This category covers:

- **Chromium** — the open-source browser project behind Chrome
- **Firefox** — Mozilla's open-source browser
- **Vivaldi** — a highly customizable browser

Browse the workflows below. To deploy one, see [How it works](/hub/getting-started/how).

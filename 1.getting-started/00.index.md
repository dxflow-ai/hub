The dxflow Hub is a curated catalog of production-ready workflows for scientific computing, data science, and engineering. Each workflow is a Docker Compose definition that runs on Docker, Podman, Singularity, or Apptainer through dxflow's unified CLI, API, and web console.

Deploy any workflow with a single command:

```bash
dxflow workflow create --identity <workflow-name> <compose-file>
dxflow workflow start <workflow-name>
dxflow workflow logs --live <workflow-name>
```

To understand how workflows are structured, deployed, and managed, see [How it works](/hub/getting-started/how). To browse what's available, head to a category — genomics, molecular, structural, data science, or fluid flow.

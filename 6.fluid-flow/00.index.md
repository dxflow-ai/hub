Computational fluid dynamics (CFD) tools for modeling and analyzing fluid flow, heat transfer, and multiphase phenomena. Simulate complex engineering problems with production-grade solvers.

::

## Overview

This collection includes workflows for:

- **CFD Solvers** - OpenFOAM, SU2, and other simulation engines
- **Pre-Processing** - Geometry and mesh generation tools
- **Post-Processing** - Visualization and analysis platforms
- **Multiphysics** - Coupled heat transfer and flow analysis
- **Optimization** - Aerodynamic and hydrodynamic design

## Available Workflows

Browse the workflows in this category to find CFD tools. Each workflow includes solver configurations, meshing guides, and post-processing setups.

## Deploy

Deploy CFD workflows:

```bash
# Deploy a CFD simulation
dxflow workflow create --identity <workflow-name> <compose-file>
dxflow workflow start <workflow-name>

# Follow the logs live
dxflow workflow logs --live <workflow-name>
```

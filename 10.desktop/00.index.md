Remote Linux desktops — full GUI environments streamed to your browser, backed by remote compute. These images (Openbox, a taskbar, a terminal, and a file manager over TurboVNC and noVNC) are also the base that the [browser](/hub/browser), [graphics](/hub/graphics), and [messaging](/hub/messaging) apps build on.

This category covers:

- **Ubuntu** — a familiar, widely supported desktop and the base for GUI apps
- **Fedora** — a leading-edge desktop
- **Void** — a lightweight, independent desktop
